Se publicaron nuevas versiones de Mistral Small y se resolvieron incidentes de Claude, Codex y OpenAI que afectaron a extensiones, la web y varias APIs.

Mistral Small 3.2 disponible vía API

Se lanzó el modelo Mistral Small 3.2 y, dos días después, se habilitó su endpoint mistral-small-2506. Actualiza tus llamadas para aprovechar el modelo más pequeño y rápido. Fuente

Corrección de la extensión Claude Code IDE en Windows

El problema que impedía cargar la extensión de Claude Code en Windows quedó solucionado en la versión v2.1.137. Actualiza la extensión para restablecer la funcionalidad de autocompletado y depuración. Fuente

Restablecimiento de Claude Code en la web

Una interrupción parcial del servicio web de Claude Code fue resuelta. Los usuarios pueden volver a acceder a la interfaz sin interrupciones. Fuente

Rendimiento normalizado en Codex Cloud Tasks

Los problemas de rendimiento que afectaban a Codex (Web, CLI, VS Code, App) fueron corregidos. Las tareas en la nube vuelven a ejecutarse con la latencia esperada. Fuente

Respuesta API de OpenAI con errores 404 corregida

Entre 16:05 y 16:40 PT del 8 de mayo, la Responses API devolvió errores 404 debido a un despliegue defectuoso. El rollback restauró el funcionamiento normal. Fuente

Problemas de operaciones de archivo en Claude resueltos

Elevados errores en operaciones de archivo (subidas, descargas) fueron investigados y solucionados el 8 de mayo. Las integraciones que manipulan archivos pueden reanudarse. Fuente
