ApiDelta — 11 de mayo de 2026

Resumen — Anthropic lanza Claude Platform en AWS con IAM nativo; Mistral SDK Python v2.4.5 introduce breaking changes en respuestas de workflow.

Tres puntos del día: Claude Platform activo en AWS, header beta 1M tokens retirado en Sonnet 4.5 y Sonnet 4 (peticiones >200k dan error), y Mistral Python SDK v2.4.5 rompe el parsing de eventos de workflow. Dos modelos Anthropic expiran el 15 de junio.

[INFO] Claude Platform en AWS: API completa vía endpoints AWS nativos

La API de Claude disponible en infraestructura AWS gestionada por Anthropic con facturación AWS y autenticación IAM. Superficie completa: Messages, Files, Message Batches, Managed Agents, Agent Skills, ejecución de código, tool use. anthropic-sdk-python v0.101.0 incluye hoy una nueva clase cliente AWS.

[URGENTE] Header beta 1M tokens retirado en Sonnet 4.5 y Sonnet 4

Desde el 30 de abril, el header context-1m-2025-08-07 no tiene efecto en claude-sonnet-4-5 ni claude-sonnet-4-20250514. Las peticiones que superan 200k tokens devuelven error. Migra a claude-sonnet-4-6 o claude-opus-4-6 — ventana 1M en GA sin header.

[ATENCIÓN] claude-sonnet-4 y claude-opus-4 deprecados; baja el 15 de junio

claude-sonnet-4-20250514 y claude-opus-4-20250514 deprecados el 14 de abril, bajan el 15 de junio — en 35 días. Reemplazos: claude-sonnet-4-6 y claude-opus-4-7.

[URGENTE] Mistral Python SDK v2.4.5: breaking changes en respuestas de workflow

Dos breaking changes: data.data.union(CustomTaskInProgressResponse).attributes.payload en workflows.executions.stream() modificado; data.data en workflows.events.getstreamevents() modificado. El código que parsee estas respuestas se romperá al actualizar.

[INFO] Anthropic Managed Agents: sesiones multi-agente, outcomes y webhooks en beta pública

Desde el 6 de mayo: sesiones multi-agente y Outcomes en beta pública bajo managed-agents-2026-04-01. Webhooks cubren eventos del ciclo de vida de sesiones y vaults. Refresco automático de credenciales vault para mcp_oauth.

Modelo multimodal de Google para workloads de baja latencia y alto volumen — entradas texto, imagen, vídeo, audio, PDF. Disponible en OpenRouter: google/gemini-3.1-flash-lite.
